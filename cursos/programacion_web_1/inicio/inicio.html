Programación web 1


Información general del OVI


• En la siguiente OVI se mostraran los audios, videos y documentos que se requieren para aprender el lenguaje de programación php y mysql, para ello se selecciono como ejemplo la creación y conexion de una base de datos.

Objetivos del OVI


• Establecer los pasos para la vinculación de una base de datos en una página desde PHP con opciones de consulta, edición, modificación y eliminación de datos y/o registros.

Objetivos del curso


• Crear un Objeto Virtual de Información (OVI) utilizando los lenguajes de Diseño web HTML5 y CSS3.

Tematica


• Desarrollo de CRUD ( Create, Read, Update and Delete) en PHP: mostrar la programación y los elementos necesarios para la vinculación de una base de datos en una página web PHP.
